## 任务详情

### ✅ 已完成

- [x] 项目基础架构搭建（React 18 + Vite 6 + Express 4）
- [x] 用户注册 / 登录 / JWT Token 认证
- [x] 首页鞋库展示、搜索、分类筛选
- [x] 鞋款详情页与下单流程
- [x] 订单状态机（已支付 → 已发货 → 租赁中 → 已归还 → 已完成 / 已取消）

- [x] 我的订单列表与取消功能

- [x] 管理后台 - 运营概览（鞋款数、进行中订单、累计收入、低库存预警）
- [x] 管理后台 - 订单管理（查看全部订单、修改状态）
- [x] 管理后台 - 鞋款管理（新增、编辑、上下架、库存维护）

## 使用建议

1. **本地查看**：直接打开本文件即可看到当前进度。
2. **GitHub Projects 看板**：
   - 进入仓库 https://github.com/rexii1120/shoe
   - 点击 **Projects** → **New project**
   - 选择 **Board** 模板
   - 将上方任务逐条创建为 Issue/Card，拖动到对应列
3. **维护方式**：每完成一个任务，把对应条目从 `📋 待办` 移到 `✅ 已完成`，并更新日期。
